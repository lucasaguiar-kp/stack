# Pacote de entrega do cliente

Este diretorio contem somente o material necessario para rodar o KHOMP localmente no cliente.

## Conteudo

- [deploy/client/](/Users/lucasaguiar/Downloads/stack-pbx/client-dist/deploy/client)
- [infra/](/Users/lucasaguiar/Downloads/stack-pbx/client-dist/infra)

## Como usar

1. entre em `deploy/client`
2. copie `.env.example` para `.env`
3. ajuste o IP da maquina local e as tags das imagens
4. rode `docker compose up -d`

Consulte [deploy/client/README.md](/Users/lucasaguiar/Downloads/stack-pbx/client-dist/deploy/client/README.md) para os detalhes de publicacao das imagens e configuracao.
